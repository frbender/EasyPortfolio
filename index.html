Easyportfolio

EasyPortfolio is a very simple PHP-class for creating portfolios.

Download this project as a .zip file Download this project as a tar.gz file

Hallo!

Du suchst nach einer Möglichkeit, einfach Portfolios zu erstellen? Du kennste dich mit Webdesign und PHP aus, möchtest aber eigentlich nur das Design machen und den ganzen PHP-Teil außer acht lassen? Dann benutze EasyPortolio! Es ist eine einfache PHP-Klasse, die es dir ermöglicht, dein eigenes Portfolio zu designen. Lade dir die Klasse noch heute herrunter und probiere sie aus. Du wirst es nicht bereuen!

Benutzung

Erstelle dir am besten am Anfang eine PHP-Datei "install.php" mit dem Inhalt:

<?php
include("easyportfolio.php");
$blog = new EasyPortfolio();
$blog->createTable();
?>

Diese PHP-Datei startest du 1x um die Datenbank zu erstellen. Jetzt kannst du sie wieder löschen; sie wird nicht mehr benötigt. Jetzt ist EasyPortfolio einsatzbereit.

Funktionen

getAllEnt()

gibt eine Array aus Einträgen des Portfolios zurück

Butze diese Funktion am besten so:

$posts = $blog->getAllEnt();
foreach ($posts as &$ent) {
  echo "$ent->getId() - $ent->getImage() - $ent->getSubject() - $ent->getBody()";
}

setNewEnt($subject, $body, $image)

schreibt einen neuen Beitrag in die Datenbank

Butze diese Funktion am besten so:

$blog->setNewEnt("First Image", "Thats my first image ever!!!", "img/portfolio.png");

delEnt($id)

löscht einen Eintrag aus der Datenbank

Butze diese Funktion am besten so:

$blog->delEnt(13);

delAllEnt()

löscht alle Einträge aus der Datenbank

Butze diese Funktion am besten so:

$blog->delAllEnt();

closeDB()

schließt die Datenbank

rufe diese Funktion bitte immer am Ende einer PHP-Datei auf Butze diese Funktion am besten so:

$blog->delAllEnt();
